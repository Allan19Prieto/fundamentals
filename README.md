# Fundamentals

A structured repository for practicing and mastering programming fundamentals, algorithms, data structures, and problem-solving using Python.

---

## 🎯 Goals

* Build strong foundations in programming and computational thinking.
* Practice consistently through hands-on exercises.
* Document solutions, notes, and insights.
* Create a high-quality resource for both personal growth and public reference.

---

## 📁 Project Structure

```
fundamentals/
├── src/
│   └── algorithms/        # Current focus: MIT OCW exercises
│
├── tests/                 # Unit tests (future)
├── docs/                  # Notes, study plans, references
├── .venv/                 # Virtual environment (ignored in git)
├── README.md
└── requirements.txt
```

> The structure will evolve as new topics are added (data structures, problem solving, etc.)

---

## 🚀 Current Focus

* MIT OpenCourseWare – Introduction To Algorithms (Week 01)

---

## 🧠 Philosophy

This repository is built on consistency over intensity.

The goal is not just to write code, but to:

* Understand deeply
* Solve problems efficiently
* Think like a software engineer
