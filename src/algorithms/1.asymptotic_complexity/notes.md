## Types of Notations
| Notation | Name     | Sign | Limit |
|----------|----------|------|-------|
| o(n) | Little-O  | < | Upper bound, inaccurate estimate |
| O(n)   | Big-O   |  | Upper boud, exact estimate |
| Θ(n)   | Big-Theta   | = | Lower and ipper bouns, exact estimate | 
| Ω(n)   | Big-Omega   |   | Lower bound, exact estimate unknown |
| ω(n)   | Little-Omega   | > | Lower bound, inaccurate estimate |

## Asymptotic Complexity Classes

| Notation | Name     |
|----------|----------|
| O(1) | Constant  | 
| O(log n)   | Logarithmic   | 
| O()   | Root   | 
| O(n)   | Linear   | 
| O(n²)   | Quadratic   | 
| O(n³)   | Cubic   | 
| O(kⁿ)   | Exponential   | 
| O(n!)   | Factorial   |
